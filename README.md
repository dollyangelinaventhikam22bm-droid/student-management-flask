\# Student Management System (Flask)



This is a simple Student Management System built using Flask and SQLite.



\## Features

\- Add student details

\- View all students

\- Update student year

\- Delete student record



\## Technologies Used

\- Python

\- Flask

\- SQLite

\- HTML



\## How to Run

1\. Install Flask:

&nbsp;  pip install flask

2\. Run the application:

&nbsp;  python app.py

3\. Open browser and go to:

&nbsp;  http://127.0.0.1:5000/
